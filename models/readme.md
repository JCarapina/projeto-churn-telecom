# Modelos Preditivos

O modelo final escolhido foi a **Regressão Logística**, priorizando interpretabilidade e alto recall.

Modelos testados:
- Regressão Logística (modelo final)
- Gradient Boosting (bom desempenho, menor interpretabilidade)

A escolha final foi guiada pela necessidade de explicar decisões para áreas de negócio.
